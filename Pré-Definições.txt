Sistema ERP para pequenas e médias empresas

- Sistema simples de gerenciamento para pequenas e médias empresas.
- Desenvolvimento em C++ para uso em sistemas Windows e Linux (ambiente gráfico)

Módulos:

- Controle de Usuários
- Cadastro de Pessoas + Cadastro de Empresas
  - Cadastro de Funcionarios
  - Cadastro de Cliente
- Financeiro
  - Recebimentos / Pagamentos
- Estoque
- Compras 
  - Produtos
  - Serviços
- Vendas
  - Produtos
  - Serviços
- Entregas / Visitas
- Nota Fiscal
